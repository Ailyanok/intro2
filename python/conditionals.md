# Conditionals

## Syntax
```python
if boolean_value:
  # do something
elif boolean_value:
  # do something else
else:
  # do another thing
```

## Exercises

```python
if a:
  print 0
  if b:
    print 1
if c:
  if d:
    print 2
  elif e:
    print 3
else:
  print 4
print 5
```
